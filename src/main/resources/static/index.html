Сообщения Места

Сообщения дежурных АСУ ТП уровня 1.

На сайте собраны сообщения дежурных АСУ ТП уровня 1 в период с 2007 по 2012 год. Каждое сообщение относится к определенному месту и объекту имеет дату и номер бригады. Имеется возможность добавлять удалять и редактировать сообщения, показывать сообщения для определенной даты. Вывод сообщений осуществляется постранично число сообщений на странице можно изменять. Для просмотра сообщений необходимо перейти по ссылке "Сообщения".

Сайт сделан на платфрорме Spring Boot база данных PostgreSQL. Имеется возможность расширения перечня мест и объектов и организации записи сообщений дежурного персонала для всех служб АСУ ТП.
